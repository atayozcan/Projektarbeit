Das Swift Beispiel für einen Deadlock besteht aus zwei Tasks, mit jeweils zwei Actors als Ressourcen.
Jeder Task hat eine eigene Funktion, in der er versucht sich beide Ressourcen zu holen.
Zunächst startet Task eins und holt sich erfolgreich Resource eins. Als Simulation
eines Arbeitsschrittes wird er für ein paar Millisekunden schlafen gelegt.

```
Thread 1: Starting...
Thread 1: Locked resource1
Thread 2: Starting...
Thread 2: Locked resource2
Thread 1: Trying to lock resource2...
Thread 2: Trying to lock resource1...
```
